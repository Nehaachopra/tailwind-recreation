<h1>⚡ChaiCSS </h1>
<p>Inspired by modern frameworks like <b>tailwind</b>, ChaiCSS parses class names at runtime and applies inline styles dynamically.</p>
<p>Write classes like</p>
<pre><code>&lt;div class="chai-p-2 chai-m-3"&gt;This is ChaiCSS&lt;/div&gt;</code></pre>
<p>And ChaiCSS converts them into</p>
<ul>
  <li>padding: 2px;</li>
  <li>margin: 3px;</li>
</ul>
<p>Thereby adding inline style that adds 2px padding and 3px margin!</p>
